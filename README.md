# Prompticus
 
A structured prompt library for engineers domains. Each folder contains prompts for a specific purpose.
 
## Index
 
| Folder | Purpose |
|---|---|
| [`learning/`](./learning/prompt.md) | Prompts to deeply understand new concepts from first principles |
| [`research/`](./research/prompt.md) | Prompts to explore and map scientific fields and literature |
| [`coding/`](./dev/prompt.md) | Prompts to write, review, and reason about code |
| [`ideation/`](./ideation/prompt.md) | Prompts to generate and surface improvement ideas |
